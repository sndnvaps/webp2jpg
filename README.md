# webp2jpg
convert the *.webp to *.jpg


# Download

```bash
  go get github.com/sndnvaps/webp2jpg
```

# Build && Run

```bash
  cd $GOPATH/src/github.com/sndnvaps/webp2jpg
  go build
```

# Convert the webp file to jpeg

```bash
  put the test.webp file into webp2jpg folder
  ./webp2jpg test.webp test.jpg
```
